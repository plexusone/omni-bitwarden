## [v0.1.0] - 2026-05-24

### Highlights

- Initial release of omni-bitwarden, a Bitwarden Secrets Manager provider for omnivault

### Added

- Bitwarden Secrets Manager provider implementing `vault.Vault` interface ([`5bf5612`](https://github.com/plexusone/omni-bitwarden/commit/5bf5612))
- Full CRUD operations: read, write, delete, list secrets ([`5bf5612`](https://github.com/plexusone/omni-bitwarden/commit/5bf5612))
- Multi-field support for accessing key, value, and note fields ([`5bf5612`](https://github.com/plexusone/omni-bitwarden/commit/5bf5612))
- Flexible path formats: secret key, org/key, key/field combinations ([`5bf5612`](https://github.com/plexusone/omni-bitwarden/commit/5bf5612))
- Auto-registration for `bw://` URI scheme via register package ([`697694f`](https://github.com/plexusone/omni-bitwarden/commit/697694f))
- Environment variable configuration: `BW_ACCESS_TOKEN`, `BW_ORGANIZATION_ID` ([`5bf5612`](https://github.com/plexusone/omni-bitwarden/commit/5bf5612))
- Self-hosted instance support via `BW_API_URL` and `BW_IDENTITY_URL` ([`5bf5612`](https://github.com/plexusone/omni-bitwarden/commit/5bf5612))
